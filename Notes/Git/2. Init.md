1. Creating folder

    ```
    mkdir my_project
    ```

2. Changing directory to this folder

    ```
    cd my_project
    ```

3. Initializating git inside project

    ```
    git init
    ```



# Getting rid of VIM

```
git config core.editor notepad
```

Use this command to check what editor is set up:

```
git config core.editor
```
